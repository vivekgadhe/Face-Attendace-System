#  Face Attendance System  
## The Face Attendance System is a **contactless, AI-powered attendance solution** that leverages **face recognition** to automate check‑in and check‑out processes. Built with **Python, OpenCV, and machine learning models**, it ensures accuracy, security, and efficiency compared to traditional methods.  
##Tech Stack  
- **Languages:** Python  
- **Libraries:** OpenCV, NumPy, Pandas  
- **Database:** MySQL / SQLite  
- **Optional:** Flask/Django for web interface
